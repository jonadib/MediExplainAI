# 🩺 MediExplain AI

> **Upload your medical report. Let AI explain it in plain language—not medical jargon.**

MediExplain AI is an AI-powered web application that helps people understand complex medical reports in a simple, easy-to-read format. Users can upload a PDF or image of a medical report, and Google Gemini AI analyzes the content to generate a clear summary, explain medical terms, highlight important findings, and provide educational insights in English or Bangla.

> **⚠️ Disclaimer:** This application is for educational purposes only. It does **not** diagnose diseases, prescribe treatments, or replace professional medical advice.

---

## 🌟 Features

- 📄 Upload medical reports (PDF, PNG, JPG, JPEG)
- 🤖 AI-powered report analysis using Google Gemini
- 📝 Plain-language summary of the report
- 🩸 Explanation of medical terms and test results
- ⚠️ Highlight abnormal findings
- 🌐 English and Bangla support
- ❓ Suggested questions to ask your doctor
- 💡 General lifestyle recommendations
- 📋 Copy results to clipboard
- 📥 Download AI-generated report
- 🌙 Dark mode support
- 📱 Fully responsive design
- 🔒 Privacy-first (No login, No database, No data storage)

---

## 🚀 How It Works

1. Upload your medical report (PDF or image).
2. Gemini AI extracts and analyzes the report.
3. Receive:
   - A simple summary
   - Important findings
   - Medical term explanations
   - Abnormal result highlights
   - Questions to ask your healthcare provider
   - Educational health insights
4. Download or share the generated explanation.

## 🚀 Deployment Guide

### Vercel Deployment (Recommended)
This project is configured to run serverless functions on Vercel to keep your `GEMINI_API_KEY` secure:
1. Push your code to a GitHub repository.
2. Sign in to [Vercel](https://vercel.com) and click **Add New** -> **Project**.
3. Import your GitHub repository.
4. Vercel will automatically detect Vite. Keep the default build settings.
5. Expand **Environment Variables** and add:
   - `GEMINI_API_KEY`: `your_actual_gemini_api_key`
6. Click **Deploy**.

### GitHub Pages (Static-Only Limitation)
GitHub Pages only hosts static frontend files and cannot run the backend Node/Express code (`server.ts`). Running API calls in the browser would expose your private `GEMINI_API_KEY` to the public. Therefore, **Vercel** (or Render/Heroku) is recommended so that the API key remains safe on the server side.
